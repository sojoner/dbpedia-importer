dbpedia-importer
================

A lein project to import ttl files from dbpedia into neo4j


## Usage

<pre>lein deps</pre>

<pre>lein run http://localhost:7474 /Users/BLA/Data/mappingbased_properties_en_uris_ca.ttl</pre>
